
Тестове завдання виконано на базі стандартної теми Dawn у середовищі Shopify.
Метою було створити секцію Featured Products, яка відображає рекомендовані товари з вибраної колекції, підтримує адаптивну верстку та інтегрується з кошиком без перезавантаження сторінки.


Етапи розробки:

Підготовка та кастомізація теми

Створено нову колекцію товарів у Shopify Admin і вручну визначено порядок відображення продуктів.

Підключено колекцію до секції Featured Collection на головній сторінці.

Налаштування середовища розробки

Встановлено Shopify CLI для локальної розробки.

Виконано авторизацію в обліковому записі Shopify.


Клоновано активну тему командою:

shopify theme pull


Запущено локальний сервер для тестування змін:

shopify theme dev


Розробка секції:

Модифіковано розмітку у файлі featured-collection.liquid для реалізації вимог макету.

Створено адаптивну верстку (mobile-first).

Налаштовано правильне відображення цін, кнопок “Add to cart” та підписів.

Додано стилі у відповідних CSS-файлах без використання фреймворків (Bootstrap тощо).

Тестування та фінальні правки

Перевірено роботу кошика без перезавантаження сторінки.

Виправлено стилі для різних розмірів екранів.

Проведено фінальну візуальну перевірку згідно з дизайном Figma.
